# 🏏 G BCCI Tournament Scorer

**Live cricket scoring app for our group — works on any phone/laptop, syncs in real-time!**

## 🔗 Open the App
👉 https://rahuls1071.github.io/G-BCCI/

---

## 📡 Live Scoring (Real-time across devices)
1. **Scorer** taps **🟢 Go Live** → gets a 5-letter room code (e.g. `KN4XP`)
2. Share the code in the group
3. Everyone else taps **📡 Join Room** → enters the code
4. All devices now show live score updates instantly!

> Room auto-reconnects if you close and reopen the app.

---

## 🏏 Scoring a Match
1. Tap **🗑 New** to start fresh
2. Set team names, overs, players per side, toss info
3. Add players to the pool → assign to teams
4. Pick openers + bowler → Start!
5. Tap run buttons (0–6), Extras, or Wicket to score

---

## 📊 Match History
- Tap **📊 History** to see all past matches & player stats
- History syncs automatically across all devices
- No login needed!

---

## 💡 Tips
- **↩ Undo** if you make a mistake
- **⇄ Swap** to change strike manually
- **🎳 New Bowler** at end of over (or it prompts you)
- **Add to Home Screen** for full-screen app experience
- Works offline too — syncs when back online
